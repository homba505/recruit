# CDL Recruit Bot

A Telegram bot for managing CDL driver recruitment.

## Features
- Recruiter login with unique password.
- Submit driver details (name, phone, experience, ready date, etc.).
- Forward submissions to company groups.
- Track driver status (Hired / Waiting / Rejected).
- Notify recruiters of replies.
- Admin panel (HOMBA) for managing drivers.
- Session timeout and secure login.
